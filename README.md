This game can be played on any (fairly modern) web browser. To play this game...
1. Download the index.html, styles.css and script.js files and put them in THE SAME FOLDER.
2. Double click the html file.
3. Ta-da! Good luck guessing!
(Note that this game has largely been designed with PC in mind. I'm still learning web development, so not entirely sure how well this game would fare on any non-PC device. Apologies for any inconvenience this may have caused!)
